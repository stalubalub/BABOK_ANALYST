# Short Rationale + Evidence Block

Use this pattern for every key conclusion in BABOK deliverables (Stages 1–8).

---

## Template

**Conclusion:** [One-sentence statement of the finding or decision]

**Assumptions (max 3–5):**
1. [Assumption 1]
2. [Assumption 2]
3. [Assumption 3]

**Evidence:**
| Source | Reference | Data Point |
|--------|-----------|------------|
| Stage 2 baseline | STAGE_02, Baseline Metrics | Month-end close: 5 days |
| Human input | Workshop 2026-03-15 | Sponsor confirmed budget range |
| Industry benchmark | [Source name] | Typical close: 2–3 days |

---

## Example

**Conclusion:** Automating PO workflow will reduce manual processing by 70%, meeting KPI-003.

**Assumptions:**
1. Current 45% manual PO rate is accurate (ERP audit sample, n=200)
2. Approved workflow covers ≥ 90% of PO volume
3. User adoption reaches 85% within 3 months of go-live

**Evidence:**
| Source | Reference | Data Point |
|--------|-----------|------------|
| Stage 2 | PP-003, Baseline Metrics | 540 manual POs/month of 1200 total |
| Stage 3 | PS-002 | Root cause: email-based approvals |
| Vendor demo | SAP workflow PoC 2026-02 | 92% straight-through processing in test |
